Introduction

Though my graphical work might not be relevant for this position, I decided to include it in this portfolio. I did this because despite professionally being a programmer I consider other forms of art a part of my lifelong work that has helped to shape me as well.

Applications

  • An OpenGL application that serves for cell and mitochondria visualisation. I’ve included this application as a showcase of various shaders which include semi-transparent cell membrane rendering. Next shader implements metaball technique for mitochondria visualisation based on sphere centers and radii only, i.e., no mesh data is necessary for rendering. Renders from the application are available here.

  • Batch Image Processor – A console application written in C# encapsulating various image processing methods. The application enables the user to process multiple images in parallel and is designed to be easily extensible programmatically by user-defined image processing methods. The source code.

  • Wood – The application generates different wood textures according to a selected type of tree. The source code.

  • PA166-Tester – A teaching application with GUI. My contribution to the application consisted in the implementation of parametric active contours model along with GUI for an accessible user manipulation. Active contours are a tool similar to the magnetic lasso in Photoshop used for semiautomatic edge detection. The contribution was a result of my bachelor thesis. Screenshots from the application are available here.

  • PALMS_CBIA – This is a GPU implementation of Piecewise Affine(-Linear) Mumford-Shah image segmentation method using CUDA/C++. GPU implementation was created as a counterpart to an existing CPU implementation. This is a relatively new scientific method and its GPU implementation might be considered the first of its kind. A simple benchmarking framework is included. The source code.

Games

  • Sticky Racing – A prototype of a cartoon RC cars racing game. Player can stick to the walls and perceive suburban house interior from previously unseen perspective. The game was developed in a team of three people. I was in charge of camera positioning, level design and kitchen assets modeling. Build is available at this link.

  • The Planet Full of Monsters – A playable prototype of a board game developed in a team of three people. The game is for 4 players who find themselves cast away on the Planet Full of Monsters. The goal is to collect all parts of your crashed space ship, repair it and leave the planet while fighting the monsters controlled by other players. The fighting system is based on the Goofspiel. My role in the game was the gameboard and cards illustration as well as playtesting with the rest of the members of the team. The prototype with rules in czech language is available here.

  • Five in a Row – A classical browser game with a custom AI algorithm and an infinite game board. You can try to beat the AI here. For the best experience use Google Chrome, please.

  • Couchsurfer – An early prototype of a 2D browser sidescrolling generative game. The player runs on an infinite sofa while trying to avoid various obstacles such as ashtrays, beer bottles and more. Obstacles are generated based on provided music track. We worked on this game in a team of three people and I was in charge of animation and assets design. The game is available here.
